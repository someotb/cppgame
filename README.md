# My first CPP game

I want to try code game on C++. Use only base C++ and SFML library to handle the window, sound, player

## Dependencies
`SFML` - Simple and Fast Multimedia Library
```bash
sudo apt install libsfml-dev
```

## Build

Create `build` in source directory and CMake:
```bash
mkdir build && cd build
cmake ..
clear && cmake --build . -j && ./cppgame 
```
